SkyDocs

SkyDocs est un générateur de documentation léger avec MarkDown.

Vue d'ensemble

SkyDocs est un logiciel qui prend vos fichiers MarkDown et génère un site web complet grâce à quelques tours de magie !
Vous pouvez tout configurer : le thème, les pages, les menus, ...

Fonctionnalités

  • Léger
  • Multi-système (vous avez besoin de Java)
  • Écrivez votre documentation en MarkDown
  • Utilisez le moteur de template jtwig
  • Hébergez le n'importe où : c'est du statique !
  • Thème responsive et HTML5-valid inclus
  • Générez une documentation multi-langage
  • Génération rapide
  • Et bien plus !
