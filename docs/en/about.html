Technologies

SkyDocs embeds some technologies : SnakeYAML, CommonMark, NanoHTTPD, jtwig, YUI Compressor Directory Watcher and Lunr.js.

This template has been created with the help of Bootstrap, jQuery, FontAwesome, Tocbot, isInViewport.js, AnchorJS, jQuery.print and highlight.js.
